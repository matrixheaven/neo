# Skim Slash Fuzzy Completion Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use aegis:subagent-driven-development (recommended) or aegis:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace Neo's slash command prefix-only completion with `skim`-backed fuzzy ranking and update Neo's minimum Rust version documentation to Rust 1.96.1.

**Architecture:** Keep Neo's existing completion pipeline and change only the slash branch. Collect slash candidates from the existing catalog, score them through a small Neo-local ranking adapter backed by `skim::fuzzy_matcher::skim::SkimMatcherV2`, and bypass the generic value sort for slash completions so match quality and curated command order survive.

**Tech Stack:** Rust 1.96.1, Cargo workspace dependencies, `skim = { version = "5.0.0", default-features = false }`, existing `neo-agent` interactive completion tests.

    root: &Path,
    project_trusted: bool,

    let project_prompts_dir = root.join(".neo/prompts");
    let mut completions = load_project_prompt_templates(root, project_trusted)

enum SlashMatchTier {

    Prefix,

    Fuzzy,

#[derive(Debug, Clone)]
struct ScoredSlashCandidate {
    candidate: CompletionCandidate,
    tier: SlashMatchTier,
    score: i64,
    source_rank: u8,
    original_index: usize,
}

impl ScoredSlashCandidate {
    fn sort_key(&self) -> (SlashMatchTier, std::cmp::Reverse<i64>, u8, usize, &str) {
        (
            self.tier,
            std::cmp::Reverse(self.score),
            self.source_rank,
            self.original_index,
            self.candidate.value.as_str(),
        )
    }
}

- [ ] **Step 5: Replace slash_source_candidates implementation**

Replace the existing `slash_source_candidates` function with:

```rust
fn slash_source_candidates(prefix: &str, catalog: &CompletionCatalog) -> Vec<CompletionCandidate> {
    let query = slash_query(prefix);
    let candidates = collect_slash_candidates(catalog);
    if query.is_empty() {
        return candidates
            .into_iter()
            .map(|(_, candidate)| candidate)
            .collect();
    }

    let matcher = SkimMatcherV2::default().smart_case();
    let mut scored = candidates
        .into_iter()
        .filter_map(|(original_index, candidate)| {
            score_slash_candidate(&matcher, query, original_index, candidate)
        })
        .collect::<Vec<_>>();

    scored.sort_by(|left, right| left.sort_key().cmp(&right.sort_key()));
    scored
        .into_iter()
        .map(|scored| scored.candidate)
        .collect()
}
```

fn collect_slash_candidates(catalog: &CompletionCatalog) -> Vec<(usize, CompletionCandidate)> {
    let sources = [
        (&catalog.slash_prompts, CompletionSource::SlashPrompt),
        (&catalog.prompt_packages, CompletionSource::PromptPackage),
        (&catalog.session_commands, CompletionSource::SessionCommand),
    ];
    let mut original_index = 0;
    let mut candidates = Vec::new();
    for (items, source) in sources {
        for item in items {
            candidates.push((
                original_index,
                CompletionCandidate::from_picker(item.clone(), source),
            ));
            original_index += 1;
        }
    }
    candidates
}

- [ ] **Step 7: Add scoring helpers**

Below `collect_slash_candidates`, add:

```rust
fn score_slash_candidate(
    matcher: &SkimMatcherV2,
    query: &str,
    original_index: usize,
    candidate: CompletionCandidate,
) -> Option<ScoredSlashCandidate> {
    let mut best: Option<(SlashMatchTier, i64)> = None;
    for key in slash_search_keys(&candidate.value) {
        let Some((tier, score)) = score_slash_key(matcher, &key, query) else {
            continue;
        };
        if best.is_none_or(|current| (tier, std::cmp::Reverse(score)) < (current.0, std::cmp::Reverse(current.1))) {
            best = Some((tier, score));
        }
    }
    let (tier, score) = best?;
    Some(ScoredSlashCandidate {
        source_rank: completion_source_rank(candidate.source),
        candidate,
        tier,
        score,
        original_index,
    })
}

fn score_slash_key(
    matcher: &SkimMatcherV2,
    key: &str,
    query: &str,
) -> Option<(SlashMatchTier, i64)> {
    if key == query {
        return Some((SlashMatchTier::Exact, i64::MAX));
    }
    if key.starts_with(query) {
        return Some((SlashMatchTier::Prefix, i64::MAX - 1));
    }
    if slash_key_segments(key).any(|segment| segment.starts_with(query)) {
        return Some((SlashMatchTier::SegmentPrefix, i64::MAX - 2));
    }
    matcher
        .fuzzy_match(key, query)
        .map(|score| (SlashMatchTier::Fuzzy, score))
}
```

If `clippy` flags the `best.is_none_or` comparison as too dense, replace the condition with this equivalent block:

    None => true,
};

fn slash_search_keys(value: &str) -> Vec<String> {
    let command = value.strip_prefix('/').unwrap_or(value);
    let mut keys = vec![command.to_owned()];
    if let Some(skill_name) = command.strip_prefix("skill:") {
        keys.push(skill_name.to_owned());
    }
    keys
}

fn slash_key_segments(key: &str) -> impl Iterator<Item = &str> {
    key.split(|character| matches!(character, ':' | '-' | '_' | '.' | '/'))
        .filter(|segment| !segment.is_empty())
}
